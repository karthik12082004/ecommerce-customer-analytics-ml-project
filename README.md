# E-Commerce Customer Analytics & Machine Learning

## 📌 Overview
This project analyzes customer purchasing behavior using data science techniques including exploratory data analysis, clustering, and predictive modeling.

## 🚀 Features
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- KPI analysis (Revenue, Average Purchase)
- Customer segmentation using K-Means clustering
- Predictive modeling using:
  - Linear Regression
  - Decision Tree
  - Random Forest
- Model evaluation using R², MAE, RMSE
- Feature importance analysis

## 📊 Key Insights
- Customers aged 25–35 show higher purchase activity
- Clothing category generates highest sales
- Customer segmentation identifies low, medium, and high-value users

## 🤖 Machine Learning
- Compared multiple models
- Linear Regression achieved best performance
- Feature importance shows Age as dominant factor

## ⚠️ Limitations
- Real dataset showed weak predictive correlation
- Synthetic dataset used to demonstrate ML modeling

## 🛠️ Tech Stack
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn
